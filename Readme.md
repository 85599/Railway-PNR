# Indian Railway PNR

A Python web-scraper for Indian Railways' PNR Status.

# Requirements
------------
* [Requests][1]
  - `pip install requests` 
  
* [json][2]
  - `pip install json`
  
* [tkinter] [3]
  - `pip Install tkinter`

> **Note:**

> - You must register your mobile on twilio .
> - You must obtain autorization keys from www.railwayapi.com.
